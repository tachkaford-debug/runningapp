# 📱 Как запустить приложение на iPhone 15 Pro

## ✅ Сервер запущен и готов!

```
🟢 Metro Bundler работает
🟢 Порт: 8083
🟢 URL: exp://192.168.0.133:8083
🟢 Кэш очищен
```

## Инструкция по запуску

### 1️⃣ Откройте Expo Go на iPhone
Запустите приложение **Expo Go** из App Store

### 2️⃣ Отсканируйте QR-код
В терминале выше показан QR-код. Отсканируйте его через Expo Go:
- Откройте Expo Go
- Нажмите "Scan QR code"
- Наведите камеру на QR-код в терминале

### 3️⃣ Дождитесь загрузки
Приложение начнёт загружаться автоматически. Это может занять 30-60 секунд при первом запуске.

### 4️⃣ Готово!
После загрузки вы увидите главный экран приложения FitnessApp

## 🔄 Если нужно перезагрузить

### На iPhone:
- Встряхните устройство
- Выберите "Reload"

### В терминале:
- Нажмите клавишу `r`

## 📊 Что будет работать

✅ Статистика тренировок

## ⚠️ Что НЕ будет работать в Expo Go

❌ **Карты (react-native-maps)**
- На экране "Track" вместо карты будет заглушка
- На экране "Explore" также заглушка
- Это нормально для Expo Go

### Почему карты не работают?
Expo Go не поддерживает нативный модуль react-native-maps. Для работы карт нужен Development Build.

## 🗺️ Чтобы карты заработали

Создайте Development Build:

```bash
# 1. Установите EAS CLI
npm install -g eas-cli

# 2. Войдите в аккаунт Expo
eas login

# 3. Настройте проект
eas build:configure

# 4. Создайте iOS build
eas build --profile development --platform ios

# 5. Установите на iPhone
# После сборки получите ссылку для установки

# 6. Запустите с dev-client
npx expo start --dev-client
```

## 🐛 Решение проблем

### Не видно QR-кода?
Прокрутите терминал вверх - QR-код показан после запуска Metro Bundler

### iPhone не может подключиться?
Убедитесь, что iPhone и компьютер в одной Wi-Fi сети

### Ошибка при загрузке?
```bash
# Очистите кэш и перезапустите
npx expo start --clear --port 8083
```

### Приложение зависло?
- Встряхните iPhone → "Reload"
- Или нажмите `r` в терминале
