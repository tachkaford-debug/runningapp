## Как запустить на iPhone 15 Pro

### Шаг 1: Откройте Expo Go
Запустите приложение Expo Go на вашем iPhone

### Шаг 2: Отсканируйте QR-код
В терминале выше показан QR-код. Отсканируйте его через Expo Go

### Шаг 3: Дождитесь загрузки
Приложение загрузится автоматически

### Шаг 4: Перезагрузите (если нужно)
Если видите старую ошибку, перезагрузите:
- Встряхните iPhone → "Reload"
- Или нажмите `r` в терминале

## Для полного функционала с картами

Создайте Development Build через EAS:

```bash
# 1. Установите EAS CLI
npm install -g eas-cli

# 2. Войдите
eas login

# 3. Настройте
eas build:configure

# 4. Создайте build
eas build --profile development --platform ios

# 5. Установите на iPhone
# Следуйте инструкциям после сборки

# 6. Запустите
npx expo start --dev-client
```

## Полезные команды

```bash
# Перезагрузить приложение
r (в терминале)

# Открыть меню разработчика
m (в терминале)

# Очистить кэш
npx expo start --clear

# Запустить с туннелем
npx expo start --tunnel
```
