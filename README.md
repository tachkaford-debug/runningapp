# 🏃‍♂️ FitnessApp - Приложение для бега

## 🚀 Быстрый старт

```bash
# Установка
npm install

# Проверка готовности к деплою
npm run check-deploy

# Деплой на Vercel (5 минут)
npm install -g vercel

```

## ✨ Основные функции

### 🎯 Трекинг тренировок
- **GPS-отслеживание** маршрута в реальном времени
- **Статистика в реальном времени**: время, дистанция, темп
- **Сохранение маршрутов** для последующего анализа
- **Современный интерфейс** с интуитивным управлением

### 🗺️ Исследование маршрутов
- **Визуализация посещенных зон** с цветовой индикацией частоты посещений
- **История маршрутов** с возможностью просмотра прошлых тренировок
- **Статистика по зонам** и общая статистика активности

### 🏆 Система достижений
- **Уровневая система** с прогресс-барами
- **Достижения** за различные цели (дистанция, время, серии)
- **XP система** для мотивации
- **Визуальные награды** и прогресс

### 📊 Аналитика
- **Общая статистика** тренировок
- **Прогресс по времени** и дистанции
- **Лучшие результаты** и рекорды
- **Недельная и месячная статистика**

## 🚀 Установка и запуск

### Локальная разработка

1. **Клонируйте репозиторий**
   ```bash
   git clone <repository-url>
   cd FitnessApp
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Запустите приложение**
   
   Для веб-версии:
   ```bash
   npm run web
   ```
   
   Для мобильной версии:
   ```bash
   npm start
   ```
   Затем сканируйте QR-код в Expo Go приложении

### Деплой

Подробное руководство по деплою смотрите в [DEPLOYMENT.md](./DEPLOYMENT.md)

**Быстрый деплой на Vercel:**
```bash
npm install -g vercel
vercel login
vercel --prod
```

## 📋 Требования

- Node.js 16+
- Expo CLI
- iOS Simulator / Android Emulator или физическое устройство
- Expo Go приложение для тестирования

## 🔧 Настройка

### Разрешения
Приложение запрашивает следующие разрешения:
- **Геолокация** - для отслеживания маршрута
- **Фоновое обновление** - для продолжения трекинга при выключенном экране

### Конфигурация
Основные настройки находятся в:
- `constants/Colors.ts` - цветовая схема
- `app.json` - конфигурация Expo
- `package.json` - зависимости
