## ⚡ Быстрый старт

```bash
# 1. Установите зависимости
npm install

# 2. Проверьте готовность
npm run check-deploy

# 3. Соберите проект
npm run build:web

# 4. Задеплойте на Vercel
npm install -g vercel
vercel --prod
```
