# ✨ FitnessApp - Переработанное приложение

## 🎯 Что изменилось

Приложение полностью переработано в стиле **Nike Run Club** с минималистичным дизайном.

## 🎨 Дизайн

### Цветовая схема:
- **Основной**: Бирюзовый #00C896
- **Акцент**: Розовый #FF2D55  
- **Фон**: Белый/Черный (светлая/тёмная тема)
- **Карточки**: Чистые с тонкими границами

### Стиль:
- Минимализм
- Крупные цифры для статистики
- Жирные заголовки
- Плавные анимации
- Тени и скругления

## 📱 Новые экраны

### 1. Авторизация
- **login.tsx** - Вход в систему
- **signup.tsx** - Регистрация
- Минималистичный дизайн
- Валидация полей
- Интеграция с Supabase Auth

### 2. Профиль (Главная)
- **profile.tsx** - Дашборд как на фото 1
- Аватар и уровень пользователя
- Прогресс-бар до следующего уровня
- Статистика: км, тренировки, время, темп
- Достижения (бейджи)
- График активности за неделю
- Личные рекорды
- Кнопка выхода

### 3. Тренировка (Центральная)
- **run.tsx** - Экран бега с картой
- GPS трекинг в реальном времени
- Статистика: время, км, км/ч
- Большая кнопка СТАРТ (с пульсацией)
- Кнопки паузы и стопа
- Сохранение маршрута
- Карта с маршрутом (react-native-maps)

### 4. Достижения
- **achievements.tsx** - Дашборд достижений
- Карточка уровня с XP
- Список всех достижений
- Прогресс-бары для незавершённых
- Иконки и описания
- Счётчик открытых/всего

## 🗄️ База данных (Supabase)

### Таблицы:
1. **profiles** - Профили пользователей
2. **workouts** - Тренировки с GPS
3. **user_stats** - Кэш статистики
4. **achievement_templates** - Шаблоны достижений
5. **user_achievements** - Полученные достижения

### Автоматизация:
- Триггер обновляет статистику после тренировки
- RLS защищает данные
- Автосоздание профиля при регистрации

## 🚀 Запуск

### 1. Установка:
```bash
npm install --legacy-peer-deps
```

### 2. Настройка Supabase:
- Создать проект на supabase.com
- Выполнить DATABASE_SCHEMA.sql
- Скопировать URL и ключ

### 3. Переменные окружения:
```bash
cp .env.local.example .env.local
# Заполнить SUPABASE_URL и SUPABASE_ANON_KEY
```

### 4. Запуск:
```bash
npx expo start
```
