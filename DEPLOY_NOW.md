## 📋 Пошаговая инструкция

### Шаг 1: Проверка проекта (1 минута)

```bash
# Установите зависимости (если еще не установлены)
npm install

# Проверьте готовность к деплою
npm run check-deploy

# Соберите проект для проверки
npm run build:web
```

Если все прошло успешно, переходите к следующему шагу.

### Шаг 2: Создайте репозиторий на GitHub (2 минуты)

1. Перейдите на https://github.com/new
2. Создайте новый репозиторий (например, `fitnessapp`)
3. НЕ добавляйте README, .gitignore или лицензию (они уже есть)

Затем в терминале:

```bash
# Инициализируйте Git (если еще не сделано)
git init

# Добавьте все файлы
git add .

# Создайте коммит
git commit -m "Initial commit - FitnessApp v1.0.0"

# Добавьте remote (замените YOUR_USERNAME на ваш username)
git remote add origin https://github.com/YOUR_USERNAME/fitnessapp.git

# Загрузите код
git branch -M main
git push -u origin main
```

### Шаг 3: Деплой на Vercel (2 минуты)

#### Способ A: Через веб-интерфейс (Рекомендуется)

1. Перейдите на https://vercel.com
2. Нажмите **"Sign Up"** и войдите через GitHub
3. Нажмите **"Add New..."** → **"Project"**
4. Найдите и выберите ваш репозиторий `fitnessapp`
5. Vercel автоматически определит настройки:
   - Framework Preset: Other
   - Build Command: `npx expo export -p web`
   - Output Directory: `dist`
6. Нажмите **"Deploy"**
7. Подождите 2-3 минуты
8. Готово! 🎉

#### Способ B: Через CLI (Альтернатива)

```bash
# Установите Vercel CLI
npm install -g vercel

# Войдите в аккаунт
vercel login

# Деплой в production
vercel --prod
```

Следуйте инструкциям в терминале.

### Шаг 4: Проверка (30 секунд)

После деплоя вы получите URL вида:
```
https://fitnessapp-xxx.vercel.app
```

1. Откройте URL в браузере
2. Проверьте, что приложение загружается
3. Попробуйте перейти по вкладкам
4. Проверьте адаптивность (измените размер окна)

## 🔄 Как обновлять приложение

После внесения изменений в код:

```bash

git add .

# Создайте коммит
```

Vercel автоматически:
1. Обнаружит изменения
2. Соберет новую версию
3. Задеплоит её
4. Отправит вам уведомление

## 🐛 Решение проблем

### Ошибка: "Build failed"

```bash
# Очистите кэш
npx expo start -c

# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install

# Попробуйте собрать локально
npm run build:web
```

### Белый экран после деплоя

1. Откройте консоль браузера (F12)
2. Проверьте ошибки
3. Обычно помогает очистка кэша (Ctrl+Shift+R)

### Приложение работает локально, но не на Vercel

1. Проверьте логи в Vercel Dashboard
2. Убедитесь, что все зависимости в package.json
3. Проверьте переменные окружения (если используете)
